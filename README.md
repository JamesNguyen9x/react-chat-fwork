# chat-fwork

## Installation

```
$ npm install react-chat-fwork
```

## Example

``` javascript
import React, {Component} from 'react'
import {Chat} from 'chat-fwork'

class Demo extends Component {

  constructor() {
    super();
  }

  render() {
    return (<div>
      <Chat/>
    </div>)
  }
}
```
